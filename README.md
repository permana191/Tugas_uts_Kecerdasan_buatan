## 📖 Deskripsi Singkat Proyek
NEXUS.AI adalah sebuah platform *Software as a Service* (SaaS) yang dirancang sebagai Sistem Peringatan Dini Bencana (*Early Warning System*). Sistem ini menggunakan pendekatan *Machine Learning* dan *Deep Learning* untuk memprediksi **Kecepatan Angin Ekstrem (Wind Gust Speed)** berdasarkan parameter cuaca historis (suhu, kelembapan, intensitas hujan, dan tekanan udara). Aplikasi ini dilengkapi dengan antarmuka *Cyber-Glassmorphism* interaktif, komparasi algoritma secara *real-time*, dan radar cuaca global.

## 🧠 Arsitektur Algoritma yang Digunakan
Sistem ini menggunakan metode komparasi multi-algoritma untuk memastikan tingkat akurasi prediksi tertinggi:
1. **Linear Regression:** Bertindak sebagai *Baseline Model* untuk regresi linier standar.
2. **K-Means Clustering:** Digunakan dalam fase *Feature Engineering* untuk mengelompokkan pola anomali cuaca.
3. **Custom Backpropagation:** Jaringan Saraf Tiruan (JST) primitif yang dibangun murni menggunakan operasi matriks Numpy (tanpa *framework*).
4. **Artificial Neural Network (ANN - Keras):** Arsitektur *Deep Learning* multlayer perceptron dengan fungsi aktivasi non-linear.
5. **Long Short-Term Memory (LSTM):** Arsitektur sekuensial tingkat lanjut yang dioptimalkan untuk memproses data deret waktu (*time-series*) atmosferik.
